# Gochat 💬 (Work in Progress)

A highly scalable, real-time messaging platform built for performance and asynchronous communication.

_Note: This project is currently under active development._

## 🏗️ Architecture & Tech Stack

This project is built using a modern backend tech stack focused on concurrency and containerization:

- **Backend:** Go (Golang)
- **Message Broker:** RabbitMQ (for real-time, asynchronous event handling)
- **Database:** PostgreSQL (hosted on Azure)
- **Frontend:** React / JavaScript
- **Infrastructure:** Docker & Microsoft Azure

## ✨ Planned Features

- **Secure Access:** Streamlined and secure user login flows via OAuth integrations.
- **Real-Time Messaging:** Instant delivery of messages utilizing RabbitMQ for robust queueing.
- **Persistent Storage:** Chat histories and user data safely managed in PostgreSQL.
- **Cloud Deployment:** Fully containerized services orchestrated and deployed on Azure.
- **Responsive UI:** Dynamic frontend interfaces built with React.
